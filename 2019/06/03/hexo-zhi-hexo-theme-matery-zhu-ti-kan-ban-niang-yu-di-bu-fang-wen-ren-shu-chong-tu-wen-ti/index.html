hexo之hexo-theme-matery主题看板娘与底部访问人数冲突问题

打开hexo-theme-matery\layout\_partial中的footer.ejs，将本站总访问量和访客数的代码改为如下：

<% if (theme.busuanziStatistics && theme.busuanziStatistics.totalTraffic) { %>
    <span id="busuanzi_container_site_pv"></span>
    <i class="fa fa-heart"></i>
    本站总访问量 <span id="busuanzi_value_site_pv" class="white-color"></span>

<% } %>
<% if (theme.busuanziStatistics && theme.busuanziStatistics.totalNumberOfvisitors) { %>
    <span id="busuanzi_container_site_uv"></span>
    次,&nbsp;
    <i class="fa fa-users"></i>
    访客数 <span id="busuanzi_value_site_uv" class="white-color"></span> 人
<% } %>

变化就在下面两句，将后面的＜/span＞写在前面了

<span id="busuanzi_container_site_pv"></span>
<span id="busuanzi_container_site_uv"></span>

本人没学过前端的东西，格式上的小问题吧。和我好友找这个问题找了好久，终于解决了。
